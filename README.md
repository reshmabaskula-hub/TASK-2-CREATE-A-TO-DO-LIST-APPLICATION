# DESCRIPTION
The TO-DO list applications is a command-line program developed using python that helps users organize and manage their daily tasks efficiently the application allows users to add new tasks, view existing tasks, and remove completed tasks. tasks are stored in a list during program execution, making it easy to track and manage activities  
This project demonstrates fundamental python concept such as lists,loops,functions,conditional statements,and exception handling while providing a pratical productivity tool
# OBJECTIVE
The objective of this project is to create a simple and user-friendly task management application that enables users to:
Add tasks to a list
View all saved tasks 
Remove completed tasks 
Manage daily activities efficiently

# CONCLUSION
The TO-DO list application is a simple yet effective task management system developed using python it allows users to add,view and remove tasks through a command line interface the project successfully demonstrates essential programming concepts such as lists,loops,functions,and error handling it serves as a pratical productivity tool and a strong foundation for building more advanced task management applications in the future
